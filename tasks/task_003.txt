# Description: Implement Kafka producer to serialize and publish raw GitHub webhook events to the 'issues.raw' topic.
# Details:
1. Add Kafka producer to the webhook receiver using aiokafka (v0.8.1+):
```python
from aiokafka import AIOKafkaProducer
import json

producer = AIOKafkaProducer(
    bootstrap_servers='redpanda:9092',
    value_serializer=lambda v: json.dumps(v).encode('utf-8')
)

async def publish_event(event_type: str, payload: dict):
    await producer.send_and_wait(
        'issues.raw',
        value=payload,
        key=str(payload.get('id', '')).encode()
    )
```

3. Implement error handling and retry logic for producer failures
4. Add monitoring for Kafka producer metrics
5. Ensure at-least-once delivery semantics
6. Implement graceful shutdown to flush pending messages
